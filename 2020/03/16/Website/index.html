電商網站專案

任職網站電商網頁設計師5-6年，建立前台頁面版型設計與活動Banner、企劃主題與採訪報導，相關活動行銷與相關影片剪輯企劃製作，這個電商網站經營的形式在當時很特別，主要服務對象為沙龍連鎖店舖的設計師，提供他們販售產品的另一個渠道，透過ERP帶入設計師分潤資料，累計統計店販業績。
  • 技術使用：
    • 繪圖軟體：photoshop、illustrator
    • 影片剪輯：after effect
    • 網頁技術：bootstrap、jq、vscode、html、css

沙龍髮廊連鎖店經營電商網站的契機

過去傳統顧客至店面消費，設計師推薦相關髮品販售的商業模式，客人需要到店消費，才能購買到沙龍品牌獨家代理的髮品。

現象觀察-自媒體崛起


而過去10年，社群平台興起、網路直播帶貨盛行，自媒體蓬勃發展之下，設計師開始透自媒體販售相關髮品，自行進貨公司代理的產品，並在「蝦皮」、「FB」、「IG」、「PCHOME」上販售。開始造成以下問題

開闢電商通路，勾稽設計師網路販售販售產品的業績

因此，總公司內部討論後，開啟了EC電商服務，為提供更便利的購買方式，與為髮型設計師線上販售產品的需求。
髮品部門規劃成立電商網站平台，並透過經營社群，以產品作為銷售目標進行內容經營，除了讓一般消費者也能透過網站認識「專業沙龍級」髮品，也能盡量杜絕設計師私下販售產品的問題。





網站規劃重點

(網站架構圖)

  1. 產品內容經營與特惠活動

    • 網站以介紹產品內容為主，將公司代理的品牌內容做經營與推廣
    • 優惠活動推廣
    • 建立通路的選品規則，區分相關競品市場
    • 產品服務品質保證（從商品到運送、付款、退貨、使用…等相關服務）

  2. 社群推廣服務：

    • 使連鎖分店的設計師有更多的商品素材，可以在他們個人的自媒體上分享

  3. 培養KOL:

    • 以電商平台作為媒體採訪公司內部設計師，並製作成行銷內容
    • 運用facebook成立內部社團，邀請設計師使用產品並投稿髮型作品，在網路上刊登廣告貼文與廣告素材，以口碑行銷的方式推動網路購買。

  4. 拓展官方通路

    • 透過自營電商在網站上埋點，偵測廣告訊息，觀察TA需求，並透過廣告效益反饋，製作廣告素材
    • 在各大電商平台開店，統一視覺形象

  5. 營造趨勢

    • 網站平台規劃各式產品主題，提出最新且時尚的髮品趨勢，曝光相關素材並帶入商品話題

目標：整合品牌、設計師及消費者達成多贏的局面

因為契機開始重新盤點品牌需求，並且整合品牌價值，建立電商平台除了解決「販售通路不一水貨猖獗」的問題外，更可透過強化沙龍連鎖店的競爭優勢，整合品牌、設計師與消費者，達成以下目標：

  • 提高品牌曝光及競爭力

    • 透過網站及社群經營,能有效提升品牌知名度及形象。
    • 可主動掌握市場趨勢,及時推出符合消費者需求的髮品。

  • 強化與設計師的合作關係、提升整體營運效率

    • 邀請設計師參與內容創作及KOL合作,能凝聚團隊向心力。
    • 多元的銷售管道,促進雙方業績成長。
    • 電商數據分析,制定更精準的行銷策略。
    • 降低通路成本,提升整體獲利空間。



  • 優化購買體驗/促進設計師推動網路購物

    • 建立統一的價格及服務標準,消除消費者的困擾
    • 透過網路銷售,提供消費者更便利的購買方式。
    • 掌握各通路秩序、控制價格競爭。
    • 降低水貨及仿冒產品的問題,維護消費者權益。

產品內容行銷文案撰寫


在產品內容曝光上，與主管討論產品成分與內容、所期望呈現的視覺形象，是我每天主要的工作。除了代理品牌外，公司也有委託大廠生產自有品牌，我設計了其中一個「鉑金光燦」修護系列的髮品。從瓶器挑選、設計logo、瓶器標示排版、打樣、填充測試、生產、驗貨，從零到有規劃上線的過程。後續也繼續跟主管討論該系列商品的產品內容呈現，網路視覺的規劃、拍攝、文案撰寫。

期望透過自有商品增加商品販售利潤，同時也配合各大研發廠，研發根據分店客單項目趨勢所需要的燙染藥水及系列產品，提供獨家的商品以及商品使用教育，使髮型設計師能夠將相關技術應用創作他們的作品。

而這個系列後來也成為公司熱銷的系列商品，除了洗髮精、護髮乳、免沖洗護髮、也延伸商品範圍至燙染藥水、護髮保養劑，搭配染燙護流程中、需維持髮質的相關需求商品相關設計，曾經參與全程的我感到與有榮焉。






影片素材與製作

在本站氛圍營造上，期待客戶能感受到時尚美感與專業髮品力，除了商品內容經營，我會定期協助拍攝相關模特兒素材、制定主活動題、採買攝影素材，推廣產品。

有時候會由我直接企劃拍攝主題、採訪設計師，規劃拍攝內容，預算少的時候就拿著手機、和設計師討論內容並約定時間進行拍攝與剪輯，全程參與的過程，有大量預算的時候再邀請跨部門團隊協助，一起建置網站與公司品牌形象。

經營痛點

所有實驗性質的專案特色就是預算少、人力低、資源少。而專案人力配置，在我加入至離開時，最多都只有3人。1位行銷、1位客服、1位網頁設計師。為了製作出精美的畫面、有質感的內容，我需要取得產品的使用心得和相關素材。我請主管編制一些預算讓我能夠邀請KOL進行產品試用、或是加入美妝平台試用評測，來拓展口碑，但一開始很遺憾，相關提案皆被拒絕。

於是我將目標轉往公司內部的設計師們，在facebook上成立私人社團，邀請他們分享產品使用照片與客人的髮型照，作為回報，當我引用他們的照片圖片時，我也會在粉專上提及該設計師，協助他們做個人的曝光。這樣的方式讓專案成功獲得許多免費素材，也對公司的同事們行銷了這個剛開啟的網站，讓這個電商網站獲得一些關注，沒有花預算就達到某些程度的廣告效果，是令我很開心的經驗。

後來這個專案在公司內部的關注度逐漸提高，定期採訪與製作行銷內容使得沙龍門店的設計師參與度增加，網站轉換率逐漸穩定，沙龍品牌產品也透過網路逐漸廣為人知，終於漸漸達成階段性的目標。
